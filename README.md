## ðŸ“– What is Oque?

**Oque** is a powerful command-line interface (CLI) tool designed to make downloading files on Linux satisfying and efficient. Unlike standard `curl` or `wget`, Oque provides:

* **Visual Feedback:** beautiful, real-time progress bars with speed and ETA.
* **Concurrency:** Download up to **4 files at once** automatically.
* **Media Support:** Built-in integration with `yt-dlp` for video/audio.
* **Repo Grabbing:** Smart handling of GitHub repository links.
